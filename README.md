#  Cake Shop Web Application (Full Stack)

This is a full-stack cake shop web application developed using Django REST Framework and React.js.

The project is divided into two separate repositories:
- Backend (API)
- Frontend (UI)

---

##  Architecture
- Frontend communicates with backend using REST APIs
- Backend handles business logic, database, and authentication
- Frontend handles UI, user interactions, and API integration

---

##  Project Repositories

###  Backend Repository
- **Tech:** Python, Django REST Framework
- **Features:** Cake listing, categories, orders, order cancellation
- **Database:** MySQL / SQLite  
   GitHub: https://github.com/yourusername/cakeshop-backend

---

###  Frontend Repository
- **Tech:** React.js, HTML, CSS, JavaScript
- **Features:** Cake display, order UI, responsive design
- **API Integration:** Connected with Django REST APIs  
   GitHub: https://github.com/yourusername/cakeshop-frontend

---

##  Key Features
- Full-stack architecture
- REST API integration
- Responsive UI
- Clean and modular code structure

---

##  Purpose
This project was built to understand real-world full-stack development with separate frontend and backend architecture.
